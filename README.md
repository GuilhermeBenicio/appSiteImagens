O projeto é uma rede social focada exclusivamente em posts de fotos, e toda as funcionalidades e verificações do site estão sendo realizadas através de uma API em desenvolvimento, conceitos do React.JS estão sendo utilizados como hooks, componentes, react-router-dom, etc. Segue abaixo algumas funcionalidades da aplicação:

- Cadastro de Usuário
- Login
- Post de fotos
- Feed com todas as fotos
- Feed só com fotos de quem você segue
- Seguir usuario
- Editar comentário
- Comentários nos posts
- Deletar post
- Feed usuários que segue
